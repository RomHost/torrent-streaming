Project Description

Netix is an online video streaming wizard that allows you to stream entire movies from a torrent file as small as a few kilobytes.

Format supporter

  • Mp4 (AAC/H.264)
  • Mp3
  • Ogg (Vorbis/Theora)
  • Webm (Vorbis/VP8)
  • Wav
